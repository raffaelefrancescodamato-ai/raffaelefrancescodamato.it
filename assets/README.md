# Sito statico pronto per GitHub

Il progetto e gia pronto all'uso: non serve alcun server locale o build. I file che contano sono quelli gia presenti in questa cartella.

- Carica su GitHub `index.html`, la cartella `assets/` e le immagini `AquaMind.png` e `rdditalia.png`.
- Per vedere il sito in locale basta aprire `index.html` con il browser.
- Per GitHub Pages: in Settings > Pages scegli la branch principale e come cartella `/` (root). Dopo il deploy il sito sara raggiungibile all'URL indicato da GitHub.
- Se preferisci usare la cartella `docs/`, copia qui dentro gli stessi file e punta Pages a `docs`.

Nessun comando da lanciare, e gia tutto pronto al copia e incolla.
